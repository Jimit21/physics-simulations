Theory -- Projectile Motion

A projectile is any object thrown with an initial velocity that then follows a path determined entirely by gravity (air resistance is neglected). The resulting path is a parabola.

At 45 degrees, the range is maximized for a given speed. The flight time depends only on the vertical component of velocity. The horizontal velocity remains constant throughout the flight.

Key equations: Horizontal: x = v0 cos(theta) t | Vertical: y = v0 sin(theta) t - (1/2)g t squared | Time of flight: T = 2 v0 sin(theta) / g
